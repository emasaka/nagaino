nagaino API documentation

expansion

/api/v0/expand (GET)

You can call this method to expand multiple shortened URLs recursively. Note that the length of request URL is limited.

Parameters

q
one or more shortened URLs
format
one of the following:
  • json_simple: JSON hash, which keys are shortened URLs, and values are expaned URLs
  • json: JSON hash, which keys are shrtened URLs, and values are full information
  • json_array: JSON array of full information
  • html: HTML

/api/v0/expandText (POST)

You can call this method to expand multiple shortened URLs recursively.

Parameters

shortURLs
text data which each line is short URL.
format
one of the following:
  • json_simple: JSON hash, which keys are shortened URLs, and values are expaned URLs
  • json: JSON hash, which keys are shrtened URLs, and values are full information
  • json_array: JSON array of full information
  • html: HTML
